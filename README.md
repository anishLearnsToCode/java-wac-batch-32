````text

````

- WAP to create a function that calculates the Binomial Coefficient `nCr`. Where:
````text
nCr = n! / (r! * (n-r)!)
4C0 = 4! / 0! 4! = 1
3C2 = 3! / 2!*1! = 6 / 2 = 3
````

- WAP to create a function that calculates the Permutation value `nPr`. Where
````text
nPr = n! / (n-r)!
5P0 = 5! / 5! = 1
6P3 = 6! / 3! = 6*5*4 = 120
````
